Hostel Room Allocation System (8086 Assembly Language)

The Hostel Room Allocation System is a menu-driven application developed in 8086 Assembly Language using EMU8086/MASM. It simulates a basic hostel management system that allows users to view room details, book vacant rooms, vacate occupied rooms, and manage tenant information through a simple text-based interface.

The system maintains records for 10 hostel rooms (101–110), tracks occupancy status, stores tenant names, and supports different room types such as Single, Double, and Suite.

Features
View room details (number, type, status, and tenant name)
Book vacant rooms
Store and manage tenant information
Vacate occupied rooms
Display room occupancy summary
Menu-driven user interface
Formatted output with splash screen
Technologies Used
8086 Assembly Language
EMU8086 / MASM
DOS Interrupts (INT 21h)
BIOS Interrupts (INT 10h, INT 16h)
